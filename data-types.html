1. Variables and Data Types

In Python, variables are used to store data values. A variable is essentially a container for data, and each variable is associated with a data type that defines the kind of data it holds. Understanding Python's data types is crucial for performing basic operations and ensuring that the code functions as expected.

Python has several built-in data types that allow you to work with different kinds of data. The most commonly used data types are Numbers, Strings, and Lists, but Python also provides other data types such as Booleans, Tuples, Sets, and Dictionaries.

2. Numbers (Integers and Floats)

Python has two types of numeric data types: Integers and Floats.

Integers are whole numbers (positive or negative) without a decimal point. Examples include numbers like 5, -3, 100, and 0. Integers are used for counting, indexing, or for performing mathematical calculations that require whole numbers.

Floats are numbers that contain a decimal point. Examples include numbers like 5.5, -0.98, 3.14. Floats are used when precise calculations are needed, such as in measurements or divisions that result in non-whole numbers.

Python can automatically detect whether a number is an integer or a float based on the syntax.

Example: 

# Integer
a = 5

# Float
b = 3.14

print(type(a))  # Output: 
print(type(b))  # Output:

3. Strings

A String is a sequence of characters enclosed in single or double quotes. Strings in Python are used to store text data. They are one of the most commonly used data types in Python because they represent textual data in programs.

Strings can contain letters, numbers, and symbols. Python also supports multi-line strings using triple quotes (either `"""` or `'''`).

Example: 

# String with single quotes
name = 'John'

# String with double quotes
greeting = "Hello, world!"

print(type(name))  # Output: 
print(type(greeting))  # Output:
You can perform operations on strings, such as concatenation (joining strings together) and slicing (accessing parts of the string).

Example: 
# String concatenation
full_name = name + " Doe"
print(full_name)  # Output: John Doe

# String slicing
print(greeting[0:5])  # Output: Hello

4. Lists

A List is an ordered collection of items enclosed in square brackets. Lists can store multiple items in a single variable, and the items in a list can be of different data types. Lists are mutable, meaning their content can be changed after creation, which makes them highly flexible for various applications.

Lists can be used to store collections of similar or different types of data, and you can modify them by adding, removing, or changing elements.

Example: 

# List with different data types
my_list = [1, "apple", 3.14, True]

print(type(my_list))  # Output: 
print(my_list[1])  # Output: apple
Example of manipulating lists: 
# Add an item to the list
my_list.append("banana")

# Remove an item from the list
my_list.remove(3.14)

print(my_list)  # Output: [1, 'apple', True, 'banana']

5. Tuples

A Tuple is similar to a list, but unlike lists, tuples are immutable, meaning that once a tuple is created, its elements cannot be changed, added, or removed.

Tuples are used when you need to store a collection of data but don’t want that data to be modified after its creation. They are often used in cases where a fixed collection of items is required.

Example: 

# Tuple with different data types
my_tuple = (1, "apple", 3.14, True)

print(type(my_tuple))  # Output:

6. Booleans

The Boolean data type is used to represent one of two values: True or False. It is often used in conditional statements or loops to check if a condition is met.

Booleans are used in comparisons and logical operations. They are essential for controlling program flow, such as whether to execute certain blocks of code based on a condition.

Example: 

# Boolean values
is_python_fun = True
is_raining = False

print(type(is_python_fun))  # Output:

7. Sets

A Set is an unordered collection of unique elements. Sets are useful when you want to store data without duplicates. Sets are also mutable, so you can add or remove items after the set is created.

One key feature of sets is that they automatically eliminate duplicate elements. If you try to add an element that is already in the set, it will not be added.

Example: 

# Set with unique elements
my_set = {1, 2, 3, 4, 5}

print(type(my_set))  # Output: 
print(my_set)  # Output: {1, 2, 3, 4, 5}
Example of eliminating duplicates: 
# Adding duplicate elements
my_set.add(3)
print(my_set)  # Output: {1, 2, 3, 4, 5} (No duplicates)

8. Dictionaries

A Dictionary is an unordered collection of key-value pairs. Each key must be unique, and each key is associated with a value. Dictionaries are used to store data in a way that allows quick access to values using the corresponding keys.

Example: 

# Dictionary with key-value pairs
my_dict = {"name": "John", "age": 25, "city": "New York"}

print(type(my_dict))  # Output: 
print(my_dict["name"])  # Output: John
Dictionaries are useful when you need to store data that has a logical relationship, such as person’s information or mapping items to their prices.

9. Type Casting

Type Casting allows you to convert one data type to another. For example, you can convert a string to an integer or a float to an integer using Python's built-in functions. Type casting is useful when you need to ensure that variables are of the correct type for certain operations.

Example: 

# Converting a string to an integer
num_str = "10"
num_int = int(num_str)
print(type(num_int))  # Output: 

# Converting a float to an integer
float_num = 3.14
int_num = int(float_num)
print(int_num)  # Output: 3
